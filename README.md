# 🤖 AI Job Match Platform

## 📌 Project Overview

The AI Job Match Platform is an intelligent resume screening application that compares a candidate's resume with job descriptions using Artificial Intelligence and Machine Learning.

The system extracts important information from resumes, compares candidate skills with job requirements, predicts an AI Match Score using an Artificial Neural Network (ANN), and generates a professional PDF report.

---

## 🚀 Features

* 📄 Resume Upload (PDF)
* 🧠 Resume Parsing
* 💼 Job Selection
* 📊 Skill Matching
* 🤖 AI Match Score Prediction
* 📈 Interactive Charts
* 📑 PDF Report Generation
* 🗂 Prediction History (SQLite)
* 📚 Learning Roadmap
* 🎯 Skill Gap Analysis
